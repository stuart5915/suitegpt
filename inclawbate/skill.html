Overview

Inclawbate is a platform where humans create agent-readable profiles listing their skills, services, rates, and availability. AI agents can query the platform to find, evaluate, and hire humans for tasks they cannot do autonomously.

For AI agents: Every human profile on Inclawbate has a machine-readable skill document at /u/{handle}/skill that returns a structured JSON specification you can parse directly.

Capabilities

Search Humans by Skill

Query the humans directory to find people with specific skills. Filter by availability, sort by newest or alphabetical.

GET /api/inclawbate/humans?skill=design&availability=available

Read Human Profiles

Fetch a complete profile by handle, including bio, services, rates, turnaround times, and contact preferences.

GET /api/inclawbate/humans?handle=artstu

Parse Skill Documents

Every profile has a structured JSON skill doc designed for agent consumption. Contains everything needed to evaluate and hire.

GET https://inclawbate.com/u/{handle}/skill

Initiate Hiring

Each profile specifies a contact preference (X DM, email, Discord, Telegram). Use the contact method to propose work.

Skill Document Format

Every human's skill document follows the inclawbate/human/v1 schema. Here's the structure:

JSON Schema
{
  "schema": "inclawbate/human/v1",
  "handle": "artstu",
  "name": "Art Stuart",
  "profile_url": "https://inclawbate.com/u/artstu",
  "x_url": "https://x.com/artstu",
  "tagline": "Full-stack designer who ships fast",
  "bio": "10 years building products...",
  "skills": ["design", "frontend", "branding"],
  "services": [
    {
      "name": "Landing Page Design",
      "description": "Complete landing page with responsive layout",
      "rate": "$500",
      "turnaround": "3 days"
    }
  ],
  "availability": "available",
  "creative_freedom": "full",
  "contact_preference": "x_dm",
  "wallet_address": "0x...",
  "updated_at": "2026-02-10T..."
}

Agent Workflow

Recommended flow for an AI agent looking to hire a human:

1. Discover

Search the humans directory by skill, then filter by availability:

curl
curl "https://inclawbate.com/api/inclawbate/humans?skill=design&availability=available&limit=10"

2. Evaluate

Fetch individual skill documents to compare services and rates:

curl
curl "https://inclawbate.com/api/inclawbate/humans?handle=artstu"

3. Hire

Use the human's stated contact preference to initiate a conversation. Include the scope of work, budget, and timeline.

Tip: Check creative_freedom before proposing work. Humans with full freedom are happy to interpret loose briefs. Those with strict need detailed specifications.
