ğŸ“– How the System Works

The Problem:

AI coding tasks can be slow and resource-intensive. Your laptop works hard, and you can only do one thing at a time.

The Solution:

Offload heavy AI work to your beefy PC (32GB RAM!) while your laptop stays light for browsing, reviewing, and quick edits.

Key Features:

  • Multi-Window Cycling - PC runs 2-3 Antigravity windows, prompts get distributed
  • Automatic Git Sync - Changes push to GitHub, laptop auto-pulls
  • Two-Way Sync - Code on laptop OR PC, both stay in sync
  • Queue System - Send multiple prompts, they queue up and process in order
  • No Manual Steps - Everything is automated after initial setup

Typical Workflow:

  1. You're coding on laptop, need a complex change
  2. Type prompt here, hit Send
  3. Continue working on something else
  4. ~45 seconds later, changes appear in your folder
  5. Dev server hot-reloads, you see the result!
