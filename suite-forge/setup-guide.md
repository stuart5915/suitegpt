# SUITE Forge Discord Setup Guide

## Step 1: Create the Server

1. Open Discord
2. Click the **+** button on the left sidebar
3. Select **"Create My Own"**
4. Choose **"For a club or community"**
5. Name it: **SUITE Forge**

## Step 2: Create Categories & Channels

### Delete default channels first, then create:

**Category: INFO**
- #welcome (read-only for members)
- #rules (read-only)
- #announcements (read-only)

**Category: CONTRIBUTE**
- #submit (anyone can post)
- #refined-queue (read-only, bot posts here)
- #approved (read-only)
- #shipped (read-only)

**Category: REWARDS**
- #leaderboard (read-only)
- #payouts (read-only)

**Category: COMMUNITY**
- #general (anyone can chat)
- #introductions (anyone can post)
