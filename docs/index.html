What is SUITE?

SUITE is a token-powered ecosystem that lets developers build apps and get paid through microtransactions. Users deposit ETH to mint SUITE tokens, which are used across all apps in the ecosystem.

Key Concepts

Treasury - Holds ETH backing all SUITE tokens. 10% of fees go to community giving.

SUITE Token - The utility token used across all apps. Always backed by ETH in the treasury.

Apps - Micro-apps built by developers that accept SUITE for payments.

The Complete Flow

Here's what happens when you request a feature or report a bug:

  1. Discord Command - You run /addition or another command in Discord
  2. Queue Management - Your request is stored in Supabase and queued for processing
  3. Watcher Service - A Python watcher polls for new requests every 5 seconds
  4. AI Processing - Request is sent to Antigravity IDE for code generation
  5. Idle Detection - System monitors file changes ‚Äî when no changes for 30s, considers done
  6. Auto-Deploy - Changes are automatically pushed to GitHub and synced across devices
