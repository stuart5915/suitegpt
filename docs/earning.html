What's Always Free

You don't need credits for everything. These features are always free:

  • Basic app features - Tracking, logging, viewing your data
  • Creating an account - Sign up and explore
  • Browsing apps - See what's available in the ecosystem
  • Voting on polls - Help decide what gets built
  • Community access - Join Telegram and X discussions

Credits = AI Features

Credits are used for AI-powered features - things like analysis, recommendations, and smart insights. The basic functionality of every app is free.
