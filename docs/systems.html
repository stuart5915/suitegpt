Core Components

SUITE Apps

React Native mobile apps (Expo). Each app connects to shared Supabase for user auth and credit balance. Apps consume credits for AI features.

USDC Vault

Smart contract on Base network. Accepts USDC deposits, tracks LP tokens 1:1, generates yield through DeFi strategies, distributes based on user allocation.

AI Fleet

Autonomous app building system. Uses Claude/Gemini for code generation. Runs 50+ iteration cycles per app. Governed by depositor votes.

Cadence AI

Internal tooling for content automation, social posting, and development workflows. Powers work logs and scheduled content.
