## Style Notes

- All icons should be **1024x1024** for high resolution
- **No borders** - icons should fill the entire canvas
- **No text** - icons should be purely visual
- Use **soft gradients** and **glowing effects** for premium feel
- Match the SUITE dark theme aesthetic
- Icons should be recognizable at small sizes (60x60px)
