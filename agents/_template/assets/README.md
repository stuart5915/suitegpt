# Agent Assets

This folder contains media and brand assets for this agent.

## Expected Files

### avatar.png / avatar.jpg
The agent's profile picture used in:
- Factory governance dashboard
- Agent profile page
- Social media (if applicable)

**Specs:**
- Size: 400x400px minimum
- Format: PNG or JPG
- Style: Should match SUITE visual identity

### brand-guidelines.md
Voice and visual guidelines for this agent:
- Tone of voice
- Writing style
- Color palette (if different from SUITE default)
- Logo usage (if any)

### social/
If this agent creates social content:
- Template images
- Brand graphics
- Meme templates

---

## Usage

When the agent proposes content that requires visuals:
1. Reference assets from this folder
2. Or propose new assets to be created
3. Humans create/approve visual assets
